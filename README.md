# ATO
ATO and TASC Plug-in for BVE5 by Stop_Pattern  

## 詳細
東京メトロ/都営地下鉄のATO/TASCを疑似再現するプラグインです  
高橋うさお様の「メトロ総合プラグイン」との併用前提の設計です。  
ご使用の際はソースを読んで地上子を置く初心者お断り鬼畜仕様ですので分かる方だけどうぞ  

## 環境
- OS：Windows10  
- Platform：BVETrainSim v5.7 (or later)
