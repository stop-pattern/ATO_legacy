# ATO/TASC
ATO and TASC Plug-in for BVE5 by Stop_Pattern  

## feathers
ATO/TASCを疑似再現するプラグインです  
高橋うさお様の「メトロ総合プラグイン」との併用前提の設計です。  
ご使用の際はソースを読んで地上子を置く鬼畜仕様ですので分かる方だけどうぞ  

## environment
- OS：Windows10  
- Platform：BVETrainSim v5.7
